github의 default branch가 master에서 main으로 변경된지 오랜 시간이 지났지만, 귀찮다는 핑계로 아직까지 적용하지 않았었는데

이번에 main branch로 설정을 해야겠다 싶어서 VS Code에서 default branch로 main을 설정하는 방법을 남겨본다.

---

> git config --global init.defaultBranch main
