파이썬에서 가장 기본이 되는 변수와 숫자의 계산에 대해 알아보겠습니다.

## 변수
```python
var = "var" #String
var = 1 #Integer
```
다음과 같이 숫자나 문자를 집어 넣을 수 있다. 이외에도 많은 것을 집어 넣을 수 있음.

또한 문자끼리 합칠 수 있음.

```python
name = "mozi"
nick_name = "mozi" + "lee" # mozilee
```
## 계산

```python
int = 2
int + 1 # 3 - 더하기
int * 2 # 6 - 곱하기
int / 2 # 1 - 나누기
int ** 5 # 32 - 거듭제곱
int % 2 # 0 - 나머지
```
